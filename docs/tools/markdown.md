## 简介

&emsp;&emsp;Markdown 是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与 HTML 混编，也可导出 HTML、PDF 以及本身的 md 格式文件[^1]。

[^1]:[Markdown官方教程（中文）](https://markdown.com.cn/)
